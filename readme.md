# Simple mob timer

> A really minimal and basic countdown timer for mobbing sessions. 


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for macOS using [electron-packager](https://github.com/electron-userland/electron-packager) Windows and Linux coming soon.
